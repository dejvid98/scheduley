##### _Event scheduling web app_


## Requirements

- Docker

- Docker-compose

## Installation

```
    $ cd into directory

    $ docker-compose build
    
    $ docker-compose up
```
- After the containers are built and running, navigate to [localhost:3000]http://localhost:3000
